# YAVR — инструкция для агента

YAVR — menu-bar утилита голосовой диктовки для macOS (русский + английский,
IT-термины латиницей, всё распознавание on-device). Если пользователь просит
«установи и настрой YAVR» — веди его по этому файлу.

## Установка на машину пользователя

1. **Проверь окружение**: macOS 14+, Apple Silicon, Command Line Tools
   (`xcode-select -p`; если нет — `xcode-select --install`, это интерактивный
   GUI-диалог). Полный Xcode НЕ нужен. Свободное место: ~2.5 ГБ.
2. **Собери и установи**: `./scripts/install.sh` из корня репозитория.
   Скрипт делает `swift build -c release` (первый раз ~2–5 мин: тянет SPM-зависимости),
   собирает бандл `dist/YAVR.app`, копирует в `/Applications/YAVR.app`
   и запускает. Подпись ad-hoc, если нет сертификата (см. п. 4).
3. **Gatekeeper**: при жалобе «повреждено/неизвестный разработчик» —
   правый клик → «Открыть», либо `xattr -dr com.apple.quarantine /Applications/YAVR.app`.
4. **Если пользователь будет пересобирать приложение** (разработка, обновления
   из репо): сначала `./scripts/make-signing-cert.sh` — создаёт самоподписанный
   сертификат «Vox Dev Signing» в связке ключей. Без него каждая пересборка
   меняет ad-hoc подпись и macOS сбрасывает выданные разрешения. При создании
   сертификата может появиться GUI-запрос пароля (доверие) — это нормально.
   При первом codesign — «Разрешить всегда».

## Первый запуск (onboarding)

Приложение живёт ТОЛЬКО в меню-баре (иконка-волна справа сверху), в Доке его
нет — предупреди пользователя. На MacBook с чёлкой иконка может не влезть —
пусть освободит место в меню-баре.

Onboarding сам проведёт по шагам:

1. **Модель** (~570 МБ, Parakeet ASR + словарная CTC) — качается с HuggingFace
   в `~/Library/Application Support/FluidAudio/Models/`, докачивается после
   обрыва. Проверка наличия: обе папки `parakeet-tdt-0.6b-v3` и
   `parakeet-ctc-110m-coreml` существуют и не пустые.
2. **Микрофон** — системный диалог по кнопке. Если кнопка «не работает»,
   доступ ранее был отклонён: веди в Системные настройки → Конфиденциальность →
   Микрофон (кнопка в onboarding уже умная и сама ведёт туда при отказе).
3. **Универсальный доступ** — нужен для вставки текста в активное окно и для
   триггера «удерживать модификатор». Кнопка ведёт в нужный раздел. Если
   приложения нет в списке — добавить «+» → /Applications/YAVR.app. Без этого
   разрешения диктовка по кастомному сочетанию всё равно работает, но результат
   идёт только в буфер обмена.
4. **Тестовая диктовка** — удержать правый Option, сказать фразу, отпустить.

Пройти onboarding заново: меню YAVR → «Пройти настройку заново…».

## Настройка под пользователя

- **Триггер**: настройки → Основные. Либо удержание правого ⌥/⌘/⌃, либо
  кастомное сочетание (клик по полю → нажать сочетание) в режимах
  «держать и говорить» / «нажал — старт, нажал — стоп».
- **Словарь** — главное, что стоит персонализировать. Файл
  `~/Library/Application Support/Vox/glossary.json`, формат и правила — в
  README, раздел «Словарь терминов». Спроси пользователя, какие термины он
  диктует часто, и добавь их. КРИТИЧНО: в `aliases` — только точные
  кириллические написания; короткие термины, звучащие как обычные русские
  слова, оставляй без aliases (только `replacements`), иначе будут ложные
  срабатывания. Правки словаря применяются со следующей диктовки.
- После правок движка замен прогони тесты: `swift run vox-tests`
  (обычный executable-таргет: на машинах без Xcode нет XCTest).

## Диагностика

| Симптом | Причина и лечение |
|---|---|
| Иконка в меню: «Вставка недоступна…» | Нет Универсального доступа или пересборка сбросила подпись → перещёлкнуть галочку YAVR в Универсальном доступе |
| Хоткей-модификатор не срабатывает | То же: нужен Универсальный доступ |
| Кнопка микрофона молчит | Доступ был отклонён: `tccutil reset Microphone com.denissobolev.vox`, перезапустить приложение |
| `open` даёт ошибку -600 | Завис старый процесс: `kill -9 $(pgrep -x Vox); sleep 1; open /Applications/YAVR.app` |
| Термин пишется кириллицей | Добавить точное написание в `replacements` термина |
| Обычное слово превращается в термин | Убрать `aliases` термина (оставить `replacements`) или поднять `minSimilarity` до 0.75–0.8 |
| Распознавание упало / модели нет | Настройки → Основные → «Скачать заново» |

Технические заметки: bundle id — `com.denissobolev.vox` (историческое, не
менять — сбросит разрешения); исполняемый файл внутри бандла — `Vox`;
`killall Vox` убивает приложение. Логи FluidAudio идут в stderr — для
отладки запускать бинарник напрямую: `/Applications/YAVR.app/Contents/MacOS/Vox`.
