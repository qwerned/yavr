01

Иконка в меню-баре: четыре состояния

  • Иконка — «волна» из 4 столбиков, template-режим (перекрашивается системой).
  • Recording — красная анимированная волна; transcribing — пульсация; error — оранжевый бейдж.
  • Клик — меню; Start Dictation дублирует хоткей. Внизу меню — строка статуса, в т.ч. ненавязчивая индикация «вставка недоступна, только буфер» при secure input / отсутствии Accessibility.

03

Настройки — вкладка Dictionary

  • Boost terms — чипы; серым — число кириллических алиасов термина. Клик по чипу — редактирование алиасов, «×» — удалить.
  • Replacements — таблица «как слышится → как писать», варианты через запятую, редактирование по двойному клику.
  • «+ / −» внизу, поиск по словарю. Файл один: ~/Library/Application Support/Vox/glossary.json — правится и руками.

05

Onboarding первого запуска — четыре шага

  • Шаг 1: скачивание моделей (~570 MB суммарно, ASR + словарная CTC) с докачкой после обрыва — кнопка Resume вместо Download.
  • Шаги 2–3: системные разрешения. Микрофон — нативный системный диалог по кнопке; Accessibility — кнопка ведёт прямо в нужный раздел System Settings, статус обновляется сам.
  • Шаг 4: тестовая диктовка прямо в окне — убедиться, что весь пайплайн живой. Skip доступен на каждом шаге, кроме первого (без модели работать нечему).
